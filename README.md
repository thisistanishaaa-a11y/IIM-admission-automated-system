# IIM Admissions Automation System

A web-based admissions document management automation platform developed using Flask, SQLite, HTML, CSS, and JavaScript.

## Features
- Secure document upload
- File size and type validation
- Applicant information management
- Database integration

## Tech Stack
- Python
- Flask
- SQLite
- HTML/CSS
- JavaScript
Project Significance

This project provided hands-on experience in designing a real-world institutional software system involving workflow automation, database management, secure file handling, and full-stack web development.

The work strengthened my skills in software engineering, system design, and building scalable solutions for complex administrative processes.
